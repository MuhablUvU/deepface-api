# Facial Expression Recognition using VGG-Face

This project trains a facial expression recognition model using VGG-Face and DeepFace.

## Supported Emotions
- Happy
- Sad
- Neutral
- Surprise
- Angry
- Fear
- Disgust

## How to Use

1. Add your dataset in `data/your_dataset/`.
2. Run the main file:
```bash
python main.py
